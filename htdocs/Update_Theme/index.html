How to update Fable theme & Plugin

Fable version 1.1.3

  • created: 06/28/2016
  • latest update: 01/17/2017
  • by: ovatheme
  • email: ovatheme@gmail.com

General

Step 1: Download latest theme for themeforest
 
 
Step 2: Unzip package that downloaded from themeforest
 
 
Step 3: Upload changes files from new theme to your host via FTP.
You can read bellow section to know files that changed in each version. Note change file maybe in theme or plugin folder
 
 
Step 4: Update Visual composer plugin
 
a) Upload Visual composer package to your host
Upload and override js_composer.zip file to your host
 
 
b) Delete current plugin
 
 
 
c) Install Visual composer plugin
 
 
d) Active visual composer
